# Properties That Run Through Every Layer

Four things apply to every layer, so they sit across the whole framework rather than inside any single step:

## Provenance

Every output can be traced back to the evidence that justified it, including every answer given through **Ask Store** (see [[ai-pf-08-ask-store]]).

## Access Control

Who can see what, with sensitive material such as personal data or exec-only notes kept protected.

## System Ownership

One person keeps the system itself healthy, with prompts current, tags tidy, and stale evidence archived.

## Channel Routing

Each person's preferred platform and format, used both to send to them and to hear back from them, including the small circle who receive the **Decision Memo** (see [[ai-pf-07-decision-memo]]).
